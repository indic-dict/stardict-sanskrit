1. Introduction
---------------
'My Sandhi' Program is a free Java-based Sandhi Generating Program. 'My Sandhi' Program runs on any platform with Java Runtime Environment 1.5 or higher installed.


2. Install
----------
In order run 'My Sandhi' you have to install Java Runtime Environment 1.5 or higher. You can download it here(Third Link from Top, entitled JRE 5.0 Update 2 includes the JVM Technology).


After installing the Java Runtime Environment, Create a directory suitable for installation. (For example, "C:\sandhi\", if you are running a Windows machine). The installation is now finished.


3. Uninstall
------------
Locate the folder where 'My Sandhi' is installed and delete it.


4. Using 'My Sandhi' 
---------------
To start 'My Sandhi' (assuming that it is installed in "c:\sandhi\"), either
1. double-click the "sandhi.jar" file,
or
2. start a command prompt and write: "java -jar c:\sandhi\sandhi.jar".
